README

This is Blog-Lebo's analysis of the 2013 court-ordered property
reassessments across Allegheny County, Pennsylvania, on the community
of Mt. Lebanon, Pennsylvania.

For more, see:
http://bloglebo.blogspot.com/2012/02/making-sense-of-new-assessments-winners.html

REQUIREMENTS

To run the analyses in this project, you will need a Unix-like
operating system (e.g., Linux or Mac OS X) and the following software:

* The R statistical computing system
  http://www.r-project.org/

* The ggplot2 package for R
  http://had.co.nz/ggplot2/

* GNU Make (probably already installed on your computer)
  http://www.gnu.org/software/make/
